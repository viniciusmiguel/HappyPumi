# Stack Detail PR1 — Resource Detail — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax.

**Goal:** Implement the four stack resource-detail endpoints (`GetStackResources` by version, `GetStackResource` single@version, `GetLatestStackResource` single latest, `GetStackOverview`) and the console UI to surface them (resource detail drawer, per-version resources, enriched Overview).

**Architecture:** Per-version resources come from the succeeded update's persisted `Checkpoint` (new `IUpdateStore.FindByVersion`); latest resources from the stack's current `Deployment`. Single resources are located by URN within that checkpoint. The console `StackDetail` Resources/Overview tabs are extracted into focused `pages/stack/` components, with a new `ResourceDetail` drawer.

**Tech Stack:** .NET 10 / C# / FastEndpoints / EF Core + Postgres (Testcontainers in tests); React + Vite + Tailwind console.

```csharp
public StoredUpdate? FindByVersion(StackCoordinates stack, long version)
{
    var row = db.Updates.AsNoTracking().FirstOrDefault(u =>
        u.Org == stack.Org && u.Project == stack.Project && u.Stack == stack.Stack
        && u.Version == version && !u.DryRun);
```

```csharp
[Collection(HappyPumiCollection.Name)]
```

```ts
// Single resource (latest) by URN — the resource detail drawer.
export const getStackResource = (org: string, project: string, stack: string, urn: string) =>
  get<{ resource: { resource: Resource }; version: number } | null>(
    `/stacks/${org}/${project}/${stack}/resources/latest/${encodeURIComponent(urn)}`, null);

// Resources at a specific update version.
export const getStackResourcesAtVersion = (org: string, project: string, stack: string, version: number) =>
  get<{ resources: { resource: Resource }[]; version: number }>(
    `/stacks/${org}/${project}/${stack}/resources/${version}`, { resources: [], version });

// Console overview aggregation (resources + tags + referenced stacks).
export const getStackOverview = (org: string, project: string, stack: string) =>
  get<{ referencedStacks: unknown[]; resources: { resources: { resource: Resource }[] }; tags: Record<string,string> } | null>(
    `/console/stacks/${org}/${project}/${stack}/overview`, null);
```
